# JobAssistant Chrome Extension

Welcome to the JobAssistant Chrome Extension repository.

JobAssistant help’s people automate the job application process.

## Notes on <all_urls> permission setting:
This version of JobAssistant collects jobs from more locations to enhance the user experience.

The list of sources are as follows:

indeed.com
glassdoor.com
jora.com
simplyhired.com
careerjet.com

Some of these sources only provide links to job ads, as opposed to providing the job ad description on the page.
In these instances, we follow the provided link to collect the job description. This is the strictly the ONLY scenario inwhich 
we use urls outside the list above, and the existing list of third-party scripts provided in our permissinos justification field.

## How it works:

The single purpose of the JobAssistant Chrome extension is to
**Search and Apply for jobs.**

Users apply for jobs in a 3 step process:

1. Search for jobs (currently works for AUS & UK, with other countries to be added soon)
2. Generate and stylize a cover letter & resume
3. Apply for a job
